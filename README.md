Build Steps:
- `mkdir <project-name>`
- `cd <project-name>`
- `gitignore node`
- `npm init --yes`
- `npm install`
- `npm install express mongodb mongoose@4.10.8 bluebird --save`
- `touch README.md`
- `touch app.js`
